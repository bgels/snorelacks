# Description:
Snorelacks' Travel Itinerary Website! Choose a country, see its flag, weather, find famous landmarks to visit, currency rates, population, possible routes to take; all to find your perfect summer vacation!


# Install Guide:
1) Open terminal and git clone this repo through ssh (`git@github.com:stuy-softdev/snorelacks.git`) onto local computer in an accessible location
2) Navigate into cloned Repo installed using `cd snorelacks`. Then navigate into `app` directory via `cd app`
3) Create a python virtual enviornment using: `python3 -m venv venv`
4) Activate virtual enviornment by doing: `cd venv`. Then `. bin/activate`
5) Navigate to root of repo via `cd ..` (2x) to install dependencies 
6) Install required dependencies using command: `pip install -r requirements.txt` then navigate into `app` via `cd app` to prepare to launch


# Launch Codes:
- Ensure current terminal PATH is correctly routed to `snorelacks/app` if not, route to `app` using command: `cd (Directory you installed our repo into)/snorelacks/app` 
- Run command: `python3 __init__.py` or `python __init__.py`
